Soporte e Instalación

Para Android:

  1. Descarga el archivo APK desde el botón correspondiente.
  2. Abre el archivo APK en tu dispositivo Android.
  3. Si es necesario, habilita la instalación desde fuentes desconocidas (Ajustes > Seguridad > Fuentes desconocidas).
  4. Completa la instalación y abre la aplicación.

Para TVBox, SmartTV o Fire TV:

  1. Descarga el archivo APK desde el botón correspondiente.
  2. Copia el archivo APK a un dispositivo USB.
  3. Conecta el USB a tu TVBox, SmartTV o Fire TV.
  4. Usa un administrador de archivos para instalar el archivo APK.

Si tienes dudas, contáctanos por WhatsApp.
